# Gestore Biblioteca 📚

Applicazione da riga di comando per gestire una biblioteca personale, sviluppata in Python con database SQLite.

## Funzionalità
- Visualizzare tutti i libri disponibili
- Aggiungere nuovi libri al catalogo
- Cercare libri per titolo
- Registrare prestiti con data automatica
- Gestire le restituzioni

## Tecnologie
- Python 3.12
- SQLite3
- Database relazionale con 3 tabelle collegate (libri, utenti, prestiti)

## Come si usa
1. Clona il repository
2. Esegui `python3 biblioteca_setup.py` per inizializzare il database
3. Esegui `python3 biblioteca.py` per avviare il programma
